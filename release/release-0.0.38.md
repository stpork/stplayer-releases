# Release notes — 0.0.38

Release date: 25-Mar-2026 

## English
- This release is focused on stability groundwork and does not add new features or interface changes.
- Improved diagnostics for rare crashes so issues can be identified and fixed faster in future updates.
- Added performance monitoring to help catch freezes and slowdowns earlier.

### Behavior changes
- No workflow-visible behavior changes in this release.

### BREAKING CHANGES
None

### Risks / What to test
- Launch the app and confirm the main screens behave as before.
- Start playback, send the app to the background, return to it, and verify resume/history still work normally.
- Check that playback, downloads, and navigation remain stable on your usual device.
