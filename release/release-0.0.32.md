# Release notes — 0.0.32

Release date: 08-Mar-2026 

## English
- Simplified the app by removing the dedicated TV interface. The app now uses a single, unified phone interface on all devices, making it lighter and easier to maintain.
- Minor player screen visual adjustment for improved layout consistency.

### Behavior changes
- The separate TV mode setting has been removed. The app automatically adapts to your device.
- Apps running on Android TV devices will now display the standard phone interface instead of a specialized TV layout.

### BREAKING CHANGES
- The optional TV interface is no longer available. If you were using the app on a TV, you will see the phone interface instead.

### Risks / What to test
- Verify playback works normally on all device types.
- Confirm library browsing and history features function correctly.
- Test that external controls (Bluetooth, car display, etc.) continue to work as expected.

## Русский
- Упрощено приложение путем удаления отдельного ТВ-интерфейса. Приложение теперь использует единый телефонный интерфейс на всех устройствах, что делает его легче и проще в обслуживании.
- Незначительная визуальная корректировка экрана проигрывателя для улучшения компоновки.

### Изменения в поведении
- Отдельная настройка ТВ-режима удалена. Приложение теперь автоматически адаптируется к вашему устройству.
- Приложения, работающие на устройствах Android TV, теперь будут отображать стандартный телефонный интерфейс вместо специализированного ТВ-макета.

### КРИТИЧЕСКИЕ ИЗМЕНЕНИЯ
- Опциональный ТВ-интерфейс больше недоступен. Если вы использовали приложение на ТВ, вы увидите телефонный интерфейс.

### Риски / Что тестировать
- Убедитесь, что воспроизведение работает нормально на всех типах устройств.
- Подтвердите, что функции библиотеки и истории работают корректно.
- Убедитесь, что внешние элементы управления (Bluetooth, автомобильный дисплей и т. д.) продолжают работать как ожидается.
